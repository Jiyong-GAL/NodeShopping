Chapter03. 노드의 자바스크립트와 친해지기

자바스크립트 자료형
- Boolean   : true, false
- Number    : 64비트 형식의 IEEE 754값으로 정수나 부동소수
              상징적인 값 - 숫자가 아님(NaN), 무한대(Number.MAX_VALUE, Number.MIN_VALUE)
- String    : 문자열
- undefined : 값을 할당하지 않은 변수의 값
- null      : 존재하지 않는 값을 가리키는 값
- Object    : 객체(Collection), 대표적으로 Array와 Date가 있음
              객체이름.변수명 = 값 형태 또는 객체이름['변수명']=값 의 형태로 사용 가능

>> 자바스크립트에서 위 자료형들을 사용할 수 있으며, var타입을 사용함
>> var타입으로 위 자료형들을 표현하기떄문에 타입확인이 가능한 typeof연산자와 형변환이 가능한 parseInt(), parseFloat()함수도 제공됨
>> 예제 ./example/p83-84.js

자바스크립트 배열
- 여러 데이터를 하나의 변수에 담아서 사용하는 객체
- 대괄호[]를 이용해 만들고, push()함수를 이용해 값 추가 가능
- 배열의 인덱스는 0부터 시작하며, 순서가 있음
- push()    : 가장 마지막에 데이터 추가
  pop()     : 가장 마지막 데이터 삭제
  shift()   : 가장 안쪽(첫번째) 데이터 삭제
  unshift() : 가장 안쪽에 데이터 삽입
  slice()   : 해당 배열을 잘라내서 새 배열 생성
  splice(index, removeCount,([Object]))  : 지정한 index부터 removeCount만큼 삭제, 단 removeCount가 0이고 Object가 있다면 데이터 추가
>> 예제 ./example/p92.js

- 비동기 프로그래밍 방식으로 코드를 만들 때 사용
- 콜백함수를 만들어 놓고 사용하거나, 익명함수를 이용
